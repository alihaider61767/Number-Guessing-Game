# Number Guessing Game (Python)

This is a beginner-friendly Python project where the computer randomly selects a number and the user tries to guess it.

## Features
- Random number between 1 and 100
- User input validation
- Hints: Too High / Too Low
- Attempts counter

## How to Run
1. Open the folder in VS Code
2. Run this command in terminal:
   python number_guessing_game.py

## Purpose
This project is created to practice Python basics such as loops, conditions, and input handling.
